# Scouting Nederland - Competentieroos

Interactieve kwalificatiekaart (competentieroos) webapplicatie voor Scouting Nederland, ontwikkeld voor Keuzedeel Frontend.

## 🏕️ Over het Project

Deze applicatie vervangt de traditionele Excel-gebaseerde kwalificatiekaarten voor Scouting teamleiding door een moderne, interactieve webapplicatie. Teamleiders en adviseurs kunnen hun competenties digitaal scoren en direct visuele feedback krijgen in de vorm van een competentieroos (radar chart).

### Doelgroepen
* **Bevers / Welpen / Scouts (BWS)**: Teamleiding jeugdleden (5-15 jaar)
* **Explorers / Roverscouts (ER)**: Teamleiding/adviseurs jongeren (15-21 jaar)

## ✨ Features

* **Interactieve Competentieroos**: Visuele radar chart om sterke en zwakke punten in één oogopslag te zien
* **Stapsgewijze Beoordeling**: Overzichtelijke invoer per hoofdtaak met voortgangsindicator
* **Lokale Opslag**: Voortgang wordt automatisch bewaard (localStorage), geen account nodig
* **Scouting Huisstijl**: Geheel in lijn met de officiële Scouting Nederland brand guidelines
* **Responsive Design**: Mobile-first benadering, perfect bruikbaar op telefoons tijdens Scouting-overleg
* **SEO Geoptimaliseerd**: Juiste meta tags, keywords en beschrijvingen

## 🛠️ Gebruikte Technologieën

* **React (Vite)**: Snelle rendering en component-gebaseerde structuur
* **Tailwind CSS (v4)**: Styling en responsive design
* **React Router**: Client-side navigatie
* **Chart.js & React-Chartjs-2**: Genereren van de interactieve radar charts
* **Framer Motion**: Vloeiende animaties voor een premium gebruikerservaring

## 🚀 Installatie & Lokaal Draaien

1. **Clone of download de repository**

2. **Navigeer naar de projectmap**
   ```bash
   cd Scouting-Frontend/Scouting-Frontend
   ```

3. **Installeer dependencies**
   ```bash
   npm install
   ```

4. **Start de development server**
   ```bash
   npm run dev
   ```

5. **Open in de browser**
   * Navigeer naar `http://localhost:5173`

## 📊 Resultaten Exporteren

Op de resultatenpagina is het mogelijk om het overzicht op te slaan als PDF via de "Exporteer naar PDF" knop (maakt gebruik van de ingebouwde print-functionaliteit van de browser).
